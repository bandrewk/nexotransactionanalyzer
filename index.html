❗beta version❗

Some features may not work yet. Such as:
  • ❌ Nexo Booster
  • ❌ Loan tracking
  • ❌ Handling rejected / failed transactions
  • ❌ Transactions for coins where more than one network is supported
What does work?
  • ✅ Transaction tracking
  • ✅ Transaction linkage to blockchain explorer (TX linkage) for all currencies
  • ✅ Overview and graph generation
  • ✅ Coinlist, including earned in-kind
  • ✅ Portfolio value determination using exchange APIs
If you encounter any trouble feel free to send a not working .csv along with a brief description to hello@nexo-ta.com.
It will be used in a discrete manner to debug and fix the problem.
